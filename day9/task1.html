    HyperText Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript
    FORMATING TAGS
    1. Bold tag:The b tag specifies bold text without any extra importance.
    2. Italic tag:The ielement makes the surrounded text display as italic
    3. Strong tag:The strong tag is used to define text with strong importance
    4. Underline:In HTML the title tag is used to specify the title of a document, and the u tag is used to underline text.
    5. subscript:he sub tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O
    6. Deleted tag:HTML del Tag is used to mark a portion of text that has been deleted from the document
    7. Code tag:The code tag is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font
    8. Address tag:The address tag defines the contact information for the author/owner of a document or an article
    9. Cite tag:The cite tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture
    10. Keyboardtag:ition and Usage. The tag is used to define keyboard input. The content inside is displayed in the browser's default monospace font.
